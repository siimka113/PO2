Interactive Elements

In-text link

CSS selector:

.link-text

Sample code:

<p> <a class="link-text" href="index.html">Click here</a> to return to this page! </p>

Rendered element:

Click here to return to this page!

Navigation Link

CSS selector:

.navi-link

Sample code:

<nav> <a class="navi-link" href="#">Navigation Link</a> </nav>

Rendered element:

Navigation Link

Text Elements

Headings

CSS selector:

.heading

Sample code:

<h1 class="heading"> h1 heading :)</h1> <h2 class="heading"> h2 heading :)</h2> <h3 class="heading"> h3 heading :)</h3> <h4 class="heading"> h4 heading :)</h4>

Rendered element:

h1 heading :)

h2 heading :)

h3 heading :)

h4 heading :)

Paragraphs

CSS selector:

p

Sample code:

<p>paragraph paragraph paragraph paragraph paragraph paragraph paragraph</p>

Rendered element:

paragraph paragraph paragraph paragraph paragraph paragraph paragraph

Numbered Lists

CSS selector:

.number-list

Sample code:

<ol type = "1" class = "number-list">
<li> first option </li>
<li> second option </li>
</ol>

Rendered element:

  1. first option
  2. second option
